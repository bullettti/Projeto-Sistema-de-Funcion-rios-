Exercício 2 – Sistema de Funcionários
Crie um sistema para gerenciar funcionários de uma empresa, incluindo estagiários e
gerentes. O sistema deve permitir cadastrar e exibir as informações no console.
Requisitos
- Criar uma classe base Funcionario com: nome, cpf, salarioBase, getters e setters com
validação e método exibirDados().
- Criar as subclasses: Gerente (bonusAnual) e Estagiario (cargaHorariaSemanal).

- Aplicar encapsulamento, construtores e sobrecarga, herança e polimorfismo.
- Criar a classe EmpresaApp para criar e exibir os funcionários.
